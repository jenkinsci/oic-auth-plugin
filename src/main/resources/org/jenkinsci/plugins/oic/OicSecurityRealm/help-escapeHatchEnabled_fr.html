Si le serveur OpenID est indisponible, change d'emplacement ou révoque les informations d'identification du client, il n'y a aucun moyen de se connecter. Ces paramètres vous permettent de récupérer l'accès sans modifier manuellement la configuration de Jenkins sur le disque. Pour utiliser la fonctionnalité, aller à '/login' en utilisant le nom et le secret respectivement comme nom d'utilisateur et mot de passe.
